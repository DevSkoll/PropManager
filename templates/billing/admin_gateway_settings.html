Setup Instructions
{% if provider == "stripe" %}

Stripe Setup:

  1. Log in to your Stripe Dashboard.
  2. Navigate to Developers > API Keys.
  3. Copy your Publishable key and Secret key.
  4. For webhooks, set the endpoint to your webhook URL and copy the Webhook signing secret.
{% elif provider == "paypal" %}

PayPal Setup:

  1. Log in to the PayPal Developer Dashboard.
  2. Create a REST API app under Apps & Credentials.
  3. Copy the Client ID and Secret.
  4. Set the webhook URL in your app settings.
{% elif provider == "square" %}

Square Setup:

  1. Go to the Square Developer Dashboard.
  2. Create an application and get your Access Token.
  3. Copy your Location ID from the Locations page.
  4. Configure webhook subscriptions for payment events.
{% elif provider == "authorize_net" %}

Authorize.Net Setup:

  1. Log in to your Authorize.Net merchant account.
  2. Navigate to Account > Settings > API Credentials & Keys.
  3. Copy your API Login ID and generate a Transaction Key.
  4. Set the Signature Key for webhook verification.
{% elif provider == "braintree" %}

Braintree Setup:

  1. Log in to the Braintree Control Panel.
  2. Navigate to Settings > API.
  3. Copy the Merchant ID, Public Key, and Private Key.
  4. Configure webhooks under Settings > Webhooks.
{% elif provider == "plaid_ach" %}

Plaid + ACH Setup:

  1. Sign up at Plaid Dashboard.
  2. Get your Client ID and Secret from the Keys page.
  3. Set the environment (sandbox, development, or production).
  4. ACH transfers are processed via Plaid's transfer API.
{% elif provider == "bitcoin" %}

Bitcoin Setup:

  1. Generate an xpub key from your HD wallet (e.g., Electrum, Coldcard).
  2. Enter the xpub below. Never enter a private key.
  3. Choose mainnet or testnet network.
  4. Addresses are derived automatically using BIP84 (native SegWit).
{% else %}
