Template Variables

Use these variables in your subject and message templates. They will be replaced with actual values when the notification is sent.

Variable Description
{property_name}Property name
{property_address}Full property address
{temperature}Current temperature (°F)
{feels_like}Feels-like temperature (°F)
{humidity}Humidity percentage
{wind_speed}Wind speed (mph)
{wind_gust}Wind gust speed (mph)
{snow_inches}Snow accumulation (inches)
{rain_inches}Rain accumulation (inches)
{alert_type}Alert type (e.g., Snow, High Wind)
{severity}Severity level
{timestamp}Date and time of reading
Example

Subject:

{alert_type} Alert: {property_name}

Message:

Snowfall of {snow_inches}" expected at {property_name}. This is a reminder to ensure walkways, roadways, etc. are cleared for plowing services. If you need assistance please reach out. Current temperature: {temperature}°F.
